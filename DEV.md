# 一、 战术分期 (Version Strategy)

我们不求全，只求**“核心路径跑通”** + “工程亮点突出”。

🚀 Ver 1.0：核心听歌 ( The "Showcase" Version ) - 必须完成
目标：不用登录，打开就能搜，搜到就能听，听的时候有进度条和歌词。这是面试演示的底线。

核心功能：

全局播放器内核：播放/暂停、上一首/下一首、进度条拖拽、音量控制。

发现页 (Home)：推荐歌单、新歌速递（调用免登录接口）。

搜索 (Search)：搜索歌曲、展示结果。

歌词 (Lyrics)：滚动歌词（这是前端难点，容易出彩）。

截止时间：Day 8

🛠 Ver 2.0：用户系统 ( The "Full Stack" Version ) - 进阶
目标：让面试官看到你能处理“鉴权”和“状态管理”。

核心功能：

二维码登录：最简单的登录方式，不需要处理手机号验证码。

我的歌单：获取用户喜欢的音乐。

持久化：刷新页面后，登录状态还在，上次听的歌还在。

截止时间：Day 13

🎨 Ver 3.0：锦上添花 ( The "Polish" Version ) - 有时间再做
目标：UI 美化，动画效果。

功能：MV 播放、暗黑模式切换、移动端适配。

截止时间：Day 16
